Class LiturgicalDay

LiturgicalDay represents a particular moment in the liturgical calendar, like "Monday in Holy Week" or "The Eve of the Epiphany."

date

date: string
  • Defined in calendar/liturgical-day.ts:20

Dates are always stored as YYYY-MM-DD. No time or timezone information is helpful. Date math can be done using a library like js-joda.

example

const day = new LiturgicalDay(); day.date = '2020-06-25'; const [y, m, d] = day.date.split('-'); const dayDate : LocalDate = LocalDate.of(y, m, d)
